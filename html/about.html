About Hackathon Review

Welcome to Hackathon Review, the website where you can review Hackathons you have attended and see whether a Hackathon is worth going to, as rated by your peers. This website was created as a COD Programming Club project by Ridas Jagelavicius, Sean Fisher and Matas Lauzadis. This project was created to help the club learn HTML, CSS, PHP and JavaScript.
